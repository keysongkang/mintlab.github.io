# MINT Lab Website

Static website for the Materials Informatics & Theory Laboratory.

## File Organization

- `*.html`: top-level pages.
- `assets/css/`: page-specific stylesheets.
- `assets/images/site/`: shared site images such as logos and navigation art.
- `assets/images/publications/`: publication thumbnails.
- `assets/images/news/`: news thumbnails and attached news PDFs.
- `gallery/`: gallery upload area.

When adding a new publication image, place it in `assets/images/publications/` and reference it from `publications.html`.
